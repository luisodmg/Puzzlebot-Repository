# Week 2 - Puzzlebot Control

This folder contains the Week 2 ROS 2 work for the Puzzlebot, including an open-loop square controller and a simple waypoint pipeline.

## Folder Overview

- `week2_puzzlebot_control/`: source ROS 2 package (`ament_python`)
- `build/`, `install/`, `log/`: generated by `colcon build`

## ROS 2 Package

- Package name: `week2_puzzlebot_control`
- Main nodes:
  - `open_loop_square`
  - `path_generator`
  - `waypoint_follower`
- Launch file:
  - `launch/mini_challenge.launch.py`

## Build

From this `week2` folder:

```bash
colcon build --packages-select week2_puzzlebot_control
```

## Source Environment

After building (every new terminal):

```bash
source install/setup.bash
```

If you use PowerShell:

```powershell
.\install\setup.ps1
```

## Run Nodes Individually

```bash
ros2 run week2_puzzlebot_control open_loop_square
ros2 run week2_puzzlebot_control path_generator
ros2 run week2_puzzlebot_control waypoint_follower
```

## Run Mini Challenge Launch

```bash
ros2 launch week2_puzzlebot_control mini_challenge.launch.py
```

Gazebo / simulation time:

```bash
ros2 launch week2_puzzlebot_control mini_challenge.launch.py use_sim_time:=true
```

## Useful Parameters

### `open_loop_square`

- `side_length` (default: `0.50`)
- `desired_speed` (default: `0.20`)
- `desired_time` (default: `0.0`, overrides speed when > 0)
- `num_sides` (default: `4`)
- `control_rate` (default: `20.0`)

Example:

```bash
ros2 run week2_puzzlebot_control open_loop_square --ros-args -p side_length:=0.7 -p desired_speed:=0.25
```

### `path_generator`

- `waypoints`: flat list `[x1, y1, t1, x2, y2, t2, ...]`
- `publish_rate` (default: `2.0`)

Example:

```bash
ros2 run week2_puzzlebot_control path_generator --ros-args -p waypoints:="[0.5,0.0,3.0, 0.5,0.5,3.0, 0.0,0.5,4.0]"
```

### `waypoint_follower`

- `default_linear_speed` (default: `0.20`)
- `default_angular_speed` (default: `0.50`)
- `control_rate` (default: `20.0`)

Example:

```bash
ros2 run week2_puzzlebot_control waypoint_follower --ros-args -p default_linear_speed:=0.25
```
